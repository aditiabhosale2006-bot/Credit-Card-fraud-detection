# 💳 Credit Card Fraud Detection

## 📌 Overview

This project uses Machine Learning to detect fraudulent credit card transactions. The model is trained on historical transaction data and predicts whether a transaction is legitimate or fraudulent. The goal is to identify suspicious transactions and improve financial security.

## 🚀 Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature scaling
* Handling imbalanced data
* Model training and evaluation
* Fraud prediction using a trained model
* Model serialization using Pickle (.pkl)

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Pickle

## 📊 Dataset

The dataset contains credit card transaction records with anonymized features (V1–V28), along with Time, Amount, and Class. The target variable:

* 0 → Legitimate Transaction
* 1 → Fraudulent Transaction

## ⚙️ Project Workflow

1. Data Collection
